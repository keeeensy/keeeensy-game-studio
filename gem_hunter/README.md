<h1 align="center">💎 Gem Hunter 2D</h1>
<p align="center"><i>Процедурный dungeon crawler — собирай сферы, выживай, выбирайся из лабиринта</i></p>

---

**Gem Hunter 2D** — это 2D dungeon crawler на Python с Pygame. Исследуй процедурно-генерируемые лабиринты, собирай все магические сферы, чтобы открыть выход, и выживай среди врагов на 15 уровнях. Всё — графика, звук, уровни — генерируется процедурно, без единого внешнего файла.

## ✨ Возможности

- **Процедурные подземелья** — BFS-связанные лабиринты, 15 уровней (10×10 → 24×24)
- **4 типа врагов** — сложность растёт с плотностью стен, количеством врагов и сфер
- **4 ачивки** — Speedrunner, Collector, Survivor, Explorer (сохраняются в JSON)
- **Встроенный редактор уровней** — 5 кистей, зум/панорамирование, сохранение/загрузка (Ctrl+S / Ctrl+L)
- **Процедурность** — никаких внешних ассетов: графика — примитивы Pygame, звук — синтез WAV
- **Камера** — следует за игроком, тряска при получении урона, мерцание неуязвимости
- **Меню** — заставка (KEEEENSY fade), анимированное меню с частицами, выбор уровня, просмотр ачивок

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| WASD / Стрелки | Движение |
| Shift | Спринт (x1.6) |
| ESC | Пауза |
| ENTER | Пропустить интро уровня |

### Редактор уровней

| Ввод | Действие |
|------|----------|
| ЛКМ | Поставить выбранную кисть |
| ПКМ | Стереть тайл |
| Ctrl + колёсико | Зум (0.3×–3×) |
| Средняя кнопка + тащить | Панорамирование |
| Ctrl + S | Сохранить уровень |
| Ctrl + L | Загрузить уровень |
| 1–5 | Кисти: Пол / Стена / Сфера / Выход / Враг |

## 🏆 Ачивки

| Ачивка | Условие |
|--------|---------|
| Speedrunner | Пройти уровень менее чем за 30 секунд |
| Collector | Собрать 50 сфер всего |
| Survivor | Закончить уровень с < 5 HP |
| Explorer | Пройти 3 уровня |

## 📦 Установка

### Из исходников

```bash
pip install pygame
python main.py
```

### Сборка exe

```bash
pip install pyinstaller
python -m PyInstaller gem_hunter.spec
```

Результат: `dist/gem_hunter.exe`

## 🧩 Прогрессия уровней

| Уровень | Размер | Стен | Врагов | Сфер |
|---------|--------|------|--------|------|
| 1       | 10×10  | 4    | 2      | 4–7  |
| 5       | 14×14  | 12   | 4      | 9–17 |
| 10      | 19×19  | 22   | 7      | 14–27|
| 15      | 24×24  | 32   | 9      | 19–37|

> С 3-го уровня появляются группы стен 2×2. Количество стен, врагов и сфер растёт линейно с уровнем.
