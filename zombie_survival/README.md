<h1 align="center">🧟 Zombie Survival</h1>
<p align="center"><i>2D волновой survival-шутер сверху вниз</i></p>

---

Шутер с боковой прокруткой: игрок движется вертикально и стреляет горизонтально по волнам зомби. Каждая пройденная волна даёт апгрейд, а враги становятся разнообразнее.

## ✨ Возможности

- **Волновой survival** — бесконечные волны с растущей сложностью
- **4 типа врагов** — Normal, Fast, Tank, Spitter (открываются с волнами)
- **Система апгрейдов** — выбирай один из четырёх апгрейдов после каждой волны (Урон, Скорострельность, Скорость, Max HP)
- **Процедурный звук** — весь звук генерируется на лету, без внешних файлов
- **Заставка** — брендинг "Keeeensy" с анимацией появления
- **Экраны смерти, паузы и завершения волны**
- **Частицы фона** — система плавающих частиц

## 👾 Типы врагов

| Тип | HP | Скорость | Урон | Открывается | Описание |
|-----|----|----------|------|-------------|----------|
| Normal | 30 | 80 | 10 | Волна 1 | Стандартный зомби |
| Fast | 15 | 150 | 7 | Волна 2 | Хрупкий, но очень быстрый |
| Tank | 100 | 50 | 20 | Волна 3 | Медленный, живучий |
| Spitter | 20 | 60 | 5 | Волна 4 | Дальнобойный (300px) |

- Снаряды Spitter: 5 урона, скорость 200, интервал 1.5–3.0 с

## ⚡ Апгрейды

Выбираются после каждой волны (начиная с волны 2). Эффекты складываются аддитивно.

| Апгрейд | Эффект | Формула |
|---------|--------|---------|
| Урон | +25% к урону пуль | `15 × (1 + level × 0.25)` |
| Скорострельность | +20% к скорострельности | `0.3s / (1 + level × 0.2)` |
| Скорость | +15% к скорости | `200 × (1 + level × 0.15)` |
| Max HP | +20 к макс. HP | `100 + (level − 1) × 20` |

> База: урон = 15, перезарядка = 0.3 с, скорость = 200, макс. HP = 100.  
> После каждой волны игрок восстанавливает 30 HP.

## 📦 Установка

### Из исходников

```bash
pip install pygame
python main.py
```

### Сборка exe

```bash
pip install pyinstaller
pyinstaller zombie_survival.spec
xcopy /E /I sprites dist\zombie_survival\sprites
```

Результат: `dist/zombie_survival/zombie_survival.exe`

> Примечание: сборка `--onedir`. Папку `sprites/` нужно копировать вручную.
